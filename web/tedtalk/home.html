What is TeX and LaTeX?

They are typesetting systems. Typesetting means the way text is arranged and composed.

TeX was developed by Donald Knuth and released in 1978. Huge flexibility, but complex. In 1980s, Leslie Lamport used TeX as a base and added more features to create LaTeX (sometimes called "Lamport-TeX").

Today they are widely used for mathematics, computer science, economics, and so on.

You can also use online editors such as Overleaf!

Converting

LaTeX documents are saved as a .tex file. Compiling would output .dvi, .ps (postscript), or .pdf documents.

Most editors should have a button where you can press to compile your file. Or you can use the command line.

latex [filename].tex would compile [filename].tex and output [filename].dvi

pdflatex [filename].tex would compile [filename].tex and output [filename].pdf

General format (LaTeX)

        
\documentclass[options]{class}
...
\begin{document}
...
\end{document}

Preamble

The part before we begin the document is called the preamble. It defines the overall format of the document.

\documentclass[options]{class}

  • Must be put in the beginning of each file
  • [options] helps customize the document as desired.
    • This is optional.
    • If more than one, separate using commas.
    • Change font size here. Default is 10pt.
  • class indicates the appearance of the document (ex: article, letter, book, etc.).
  • Example: \documentclass[11pt,a4paper]{article}

Between \documentclass[options]{class} and \begin{document}

  • Any packages that you want to use,
  • To include title, author, and date:
    • \title{...}
    • \author{...}
    • \date{...}
      • You can put any date you want, or you can use the date which you compiled the file by typing: \date{\today}
    • Remember to put \maketitle inside the body in order to display the author, title, and date!

Body

This is in between \begin{document} and \end{document}. This is where you put whatever contents you want inside the document. If you want some text, directly type it in this section.
